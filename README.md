# CSU Notice Bot

*基于 [nonebot2](https://github.com/nonebot/nonebot2) 和 [go-cqhttp](https://github.com/Mrs4s/go-cqhttp)，使用 [csu-notice-api](https://github.com/MagicalSheep/csu-notice-api) 的中南大学校内通知 QQ 机器人。*

Demo：[Dr.Sink](https://wpa.qq.com/msgrd?v=3&uin=1351483470&site=qq&menu=yes)
